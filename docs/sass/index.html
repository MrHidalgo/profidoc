    search-key

    @function search-key($map: $object, $keys...: string, int, char... etc) { ... }

    Description

    Function returns a Boolean value, depending on the point of variable objects. It uses an additional function :

    • map-has-key() - returns whether a map has a value associated with a given key;
    • map-get() - returns the value in a map associated with the given key. If the map doesnâ€™t have such a key, returns null;

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $map

    The argument takes an object

    		Object$object
    $keys...

    Can take any number of arguments

    		String or Numberstring, int, char... etc

    Returns

    Boolean —

    return true if value find in variable $object, false/error if failed

    Example

    FUNCTION => search-key():
  @if (search-key($object, $name, $key)) {
    @content
  }
====================================================

FUNCTION => map-has-key():
  map-has-key(("foo": 1, "bar": 2), "foo") => true
  map-has-key(("foo": 1, "bar": 2), "baz") => false

====================================================
FUNCTION => map-get():
  map-get(("foo": 1, "bar": 2), "foo") => 1
  map-get(("foo": 1, "bar": 2), "baz") => null

    Throws

    • The value is not present in the object... watch [_varaible.scss]

    mixinBorder

    @mixin mixinBorder($place, $width: null, $color) { ... }

    Description

    Mixin the border shorthand property sets all the border properties in one declaration. The properties that can be set, are (in order): border-width, border-style, and border-color.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $place

    if 'all' then border: width solid color;

    		String—none
    $width

    if 'null' then border-#([top, right, bottom, left]): 1px solid color, else border-#([top, right, bottom, left]): width solid color;

    		Stringnull
    $color

    Optional. The color of the shadow. Look at CSS Color Values for a complete list of possible color values;

    		String—none

    Example

    <selector> {
  @include mixinBorder('all', #000);
  @include mixinBorder(left , null, #000);
  @include mixinBorder(left , 2px , #000);
}
